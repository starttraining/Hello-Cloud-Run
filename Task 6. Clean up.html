In this task, you delete the deployed service and the container image to avoid incurring charges. While Cloud Run does not charge when the service is not in use, you might still be charged for storing the built container image. You can delete your helloworld image using the following command: gcloud artifacts docker images delete $LOCATION-docker.pkg.dev/$GOOGLE_CLOUD_PROJECT/my-repository/helloworld When prompted to continue type Y, and press Enter. To delete the Cloud Run service, use this command : gcloud run services delete helloworld --region="REGION" When prompted to continue type Y, and press Enter.
